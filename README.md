# 🔖 Bookmark Manager Web App

This project is a **simple and efficient Bookmark Manager** built using **HTML, CSS, and JavaScript**.  
It allows users to add, view, open, and delete bookmarks, with data stored persistently using **browser localStorage**.

---

## 🚀 Features

- ➕ Add bookmarks with title and URL
- 💾 Persistent storage using `localStorage`
- 🔗 Open bookmarks in a new tab
- ❌ Delete individual bookmarks
- 🧹 Clear all bookmarks option
- 📱 Fully responsive design
- 🔒 Basic URL validation and sanitization

---

## 🛠️ Technologies Used

- **HTML5** – structure  
- **CSS3** – styling & responsive layout  
- **JavaScript (ES6)** – logic and localStorage handling
