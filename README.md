# 🔖 Bookmark Manager Web App

A **lightweight and responsive Bookmark Manager** built using **vanilla HTML, CSS, and JavaScript**.  
This application allows users to store, manage, and access bookmarks efficiently using **browser localStorage**, ensuring data persistence without a backend.

---

## 🚀 Key Features

- ➕ Add bookmarks with custom title and URL
- 💾 Persistent storage using `localStorage`
- 🔗 Open saved bookmarks in a new browser tab
- ❌ Delete individual bookmarks
- 🧹 Clear all bookmarks with confirmation
- 🔐 URL validation and input sanitization
- 📱 Fully responsive UI (desktop & mobile)
- ⚡ Fast performance (no external libraries)

---

## 🛠️ Tech Stack (Detailed)

### 🧱 Frontend Structure
- **HTML5**
  - Semantic HTML elements
  - Accessible form inputs
  - Anchor-based navigation
  - SEO-friendly markup

### 🎨 Styling & Layout
- **CSS3**
  - CSS Grid for form layout
  - Flexbox for list alignment
  - Responsive design using media queries
  - Custom buttons, hover states, and UI spacing
  - Mobile-first responsive approach
  - Box shadows and rounded UI components

### ⚙️ Logic & Functionality
- **JavaScript (ES6)**
  - DOM manipulation
  - Event handling (`click`, `submit`)
  - Input validation and sanitization
  - URL normalization (auto-adding protocol)
  - Dynamic rendering of bookmark list
  - Modular and readable function-based code

### 💾 Data Persistence
- **Web Storage API**
  - `localStorage` for saving bookmarks
  - JSON serialization & deserialization
  - Data persistence across page reloads
  - Browser-side storage without backend
